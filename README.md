# Abstract Tweaks

A standalone World of Warcraft addon providing quality-of-life improvements and tweaks.

## Features

### Auto Loot & Merchant
- **Fast Loot**: Instant item pickup without loot window delay
- **Auto Repair**: Automatically repair at vendors (with optional guild bank support)
- **Auto Sell Junk**: Automatically sell grey quality items at merchants

### Map Enhancements
- **Reveal Map**: Remove fog of war to see unexplored areas
- **Delve Pin Recoloring**: Make Delve entrance pins more visible with custom colors
  - Separate colors for normal and Bountiful Delves

### UI Scaling & Positioning
- **Quest Frame Scaling**: Adjust size and position of quest/dialogue windows
- **Group Finder Scaling**: Adjust size and position of the Group Finder window

### Fishing
- **One-Key Fishing**: Use a single keybind for both casting and hooking fish
  - Set keybind in: Interface > Keybindings > Abstract Tweaks > One-Key Fishing

### Talents
- **Talent Import Overwrite**: Add a checkbox to import dialog to overwrite current loadout instead of creating new

### Mythic+
- **Auto-Insert Keystone**: Automatically place keystones into the font

### UI Tweaks
- **Hide Bag Bar**: Hide the bag bar from the UI
- **Skip Cutscenes**: Automatically skip cinematics and movies
- **Auto-Fill Delete Confirmation**: Automatically fills in DELETE text when deleting items
- **Auto Screenshot on Achievement**: Take screenshots when earning achievements

### Sound Customization
- **Custom Whisper Sound**: Replace the default whisper notification with various preset sounds or a custom sound ID from wowhead.com/sounds

## Installation

1. Extract the `AbstractTweaks` folder to your `World of Warcraft\_retail_\Interface\AddOns\` directory
2. Restart World of Warcraft or reload your UI
3. Type `/tweaks` to open the options panel

## Commands

- `/tweaks` or `/at` - Open options panel
- `/tweaksstatus` - Show current status and diagnostics
